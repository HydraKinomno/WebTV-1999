The WebTV browser has support for many proprietary HTML tags, attributes, and features. Some of them even made their way into modern browsers by way of CSS, like gradient backgrounds, multiple font weights, and even scrolling background images.

Note that this document assumes prior basic HTML skills, particularly with legacy web development.

Backgrounds and Visuals

WebTV has many options for creating cool backgrounds and visual effects. Let's start with gradient backgrounds:

Hello! This table has a gradient background.
Neat, right? These can be done with the following code:
<table bgcolor=firstcolor gradcolor=secondcolor>   <tr><td>
    Content here
  </td></tr>
</table>
Replace "firstcolor" and "secondcolor" with the first and second colors respectively.

 Need to change the gradient's angle?
Use the "gradangle" attribute:
gradangle=45


Let's put them all together now to make a cool-looking table:

Hello! This table has a sick red-to-black gradient.
There's even a border to keep it visually separated from the rest of the content. Isn't that neat?


Looks pretty sweet, doesn't it?

WebTV also has support for scrolling background images, but only on the body tag. Use this with caution: it's been known to cause a lot of lag on real hardware, so maybe restrict its usage to Viewer-only pages.
<body background=image.gif xspeed=10 yspeed=10>


Text Effects

Now that we've messed around with table backgrounds, let's have some fun with the text! WebTV provides several ways to style your text.
Aside from the usual bold, italics, and underline, you can also have really thick text and shadowed text.

The really thick text is known as a black font face, and is usually the thickest one available in a font family.
You can create this text with the... unfortunately named <blackface> tag:
<blackface>Text in a black font face.</blackface>
This can be used to create eye-catching headings:

Functionality

WebTV has a lot of hidden tags normally used on internal pages.

<tv>

On WebTV Plus units, the <tv> tag is used in TV World to display the TV signal.
It can also be used outside of TV World; Here it is tuned to channel 3:
Note: If you can't see the TV window above, you most likely have a WebTV Classic, or you're in minibrowser mode.

This can be used to create alternate TV viewing pages or something along those lines.

<clock>

This one's a bit more useful - it displays the current date and time:
,
This can be used in several different ways, but it mainly helps to keep track of time, since WebTV doesn't have a clock anywhere else other than TV Home.

The tag has several attributes, but if used without them, it simply lists the full date:

The attributes for this tag are as follows:
Data:

Modifiers:
€ abbreviated: Abbreviates the day of the week and month: ;

These attributes can be combined together - you don't need separate clock elements for each different bit of data. Some attributes can only be used with others - for example, the ampm attribute needs to be used with time. It can't be used alone.

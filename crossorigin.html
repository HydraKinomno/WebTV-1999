Why am I here?

You're here because you tried to reload WebTV HD while it was navigated to a cross-origin page.

Because WebTV HD can't access the URL of a cross-origin page, it has no idea how to get you back to where you were.


To avoid this next time, use the options bar reload button to reload pages, rather than the button on your browser's navigation bar - this avoids reloading the entirety of WebTV HD along with it.


Note that you might also be here because you visited a WebTV HD URL that ended up going to this page.

See this page for more info on cross-origin pages.
